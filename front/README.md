### Front 환경 셋팅
 npx 명령어 사용 = node 설치 하기
 (주의) 현재 Nodejs 설치시 14버전이나 16버전 찾아서 설치하시면 됩니다 17버전 금지 
 - npx create-react-app my-app
 - cd my-app
 - npm i (open folder 시)
 - npm start

### 리엑트, 라우터
